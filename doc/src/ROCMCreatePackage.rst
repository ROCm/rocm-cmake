ROCMCreatePackage
=================

Commands
--------

.. cmake:command:: rocm_create_package

.. code-block:: cmake

    rocm_create_package(
        NAME <name>
        [DESCRIPTION <description>]
        [SECTION <section>]
        [MAINTAINER <maintainer>]
        [LDCONFIG_DIR <lib-directory>]
        [PREFIX <path>]
        [LDCONFIG]
        [HEADER_ONLY]
    )

Sets up CPack packaging.
